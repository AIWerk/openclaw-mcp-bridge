# Hostinger API MCP Server

Complete web hosting management with 119 tools for domains, hosting, emails, and website management.

## Tools Provided
Comprehensive Hostinger API integration including:
- Domain management (registration, transfer, DNS)
- Web hosting control (files, databases, SSL)
- Email account management
- Website builder tools
- VPS management
- WordPress management
- Security and backup tools
- Analytics and monitoring
- And many more hosting features...

## Authentication
Get your API token from Hostinger control panel → API Access

## Installation
Requires Node.js and npm. Installs globally via npm package.
