# Notion MCP Server

Official Notion integration for reading and managing your Notion workspace content.

## Tools Provided
- Read and search pages
- Query databases
- Create and update content
- Manage page properties
- Database operations
- Content extraction and analysis

## Authentication
Create an integration at: https://www.notion.so/my-integrations
Then get your API key from the integration settings.

## Installation
Uses npx - no installation required. Downloads automatically when first used.
