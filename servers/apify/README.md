# Apify MCP Server

Web scraping and automation platform that provides 8 essential tools for data extraction and browser automation.

## Tools Provided
- Web scraping from URLs
- Google Search integration
- Dataset management
- Actor (automation scripts) execution
- Browser automation
- Data transformation
- Content extraction
- API integrations

## Authentication
Get your API token at: https://console.apify.com/account/integrations

## Usage
This is a hosted service - no local installation required. Just configure your token and you're ready to scrape the web!
