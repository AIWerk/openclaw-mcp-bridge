# Filesystem MCP Server

Direct file system access for reading, writing, and managing files and directories.

## Tools Provided
- Read file contents
- Write and create files
- List directory contents
- File and directory operations
- Path utilities
- Text search within files

## Authentication
No authentication required - uses local file system permissions.

## Configuration
Edit the path in config.json to specify which directory the server can access.
Default: `/home/user/documents`

## Installation
Uses npx - no installation required.
