# Hetzner Cloud MCP Server

Comprehensive cloud infrastructure management with 30 tools for servers, networks, volumes, and more.

## Tools Provided
Complete Hetzner Cloud API coverage including:
- Server management (create, delete, power control, resize)
- Volume management (attach, detach, resize)
- Network and firewall configuration
- SSH key management
- Image and snapshot operations
- Load balancer management
- Floating IP management
- And much more...

## Authentication
Get your API token at: https://console.hetzner.cloud/projects → Access Tokens

## Installation
Requires Python and pip. Installs the mcp-hetzner package from GitHub.
